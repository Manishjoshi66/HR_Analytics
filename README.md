# HR_Analytics
HR Analytics involves analyzing employee data to identify workforce trends and support better HR decisions. This project explores factors such as employee attrition, job satisfaction, salary, and performance using data analysis and visualization to generate insights that help improve retention, workforce planning, and organizational productivity.
